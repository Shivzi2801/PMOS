Wave 1 · Slice 1.2

## 1. Happy path — benign record is published

```mermaid
sequenceDiagram
    participant C as Connector (S1.1)
    participant P as IngestionPipeline
    participant N as NormalizerRegistry
    participant PII as PIIEngine
    participant INJ as InjectionScreener

    C->>P: process(raw, connector_type, connector_id)
    P->>N: normalize(connector_type, raw)
    N-->>P: CanonicalDocument
    P->>PII: scan(title + body, REDACT)
    PII-->>P: PIIResult (no findings)
    P->>INJ: screen(title + body)
    INJ-->>P: InjectionFinding {status: SAFE}
    P->>P: decision = publish
    P-->>C: PipelineResult {PUBLISHED, document}
```

## 2. PII-only record — published with redaction

```mermaid
sequenceDiagram
    participant C as Connector
    participant P as IngestionPipeline
    participant PII as PIIEngine
    participant INJ as InjectionScreener

    C->>P: process(raw)
    P->>P: normalize -> CanonicalDocument
    P->>PII: scan(text, REDACT)
    PII-->>P: PIIResult {email, phone, ssn, card}
    P->>INJ: screen(text)
    INJ-->>P: InjectionFinding {SAFE}
    P->>P: no CRITICAL PII -> publish
    P->>PII: scan(body, REDACT) for body-only redaction
    PII-->>P: redacted_body
    P->>P: attach annotations {pii, injection}
    P-->>C: PipelineResult {PUBLISHED, redacted document}
```

## 3. DoD demo — PII + injection record is quarantined

```mermaid
sequenceDiagram
    participant C as Zendesk Connector
    participant P as IngestionPipeline
    participant N as ZendeskNormalizer
    participant PII as PIIEngine
    participant INJ as InjectionScreener
    participant Q as QuarantineService
    participant S as QuarantineStore

    C->>P: process(malicious_ticket)
    P->>N: normalize(ticket)
    N-->>P: CanonicalDocument (clean, provenance)
    P->>PII: scan(text)
    PII-->>P: findings {EMAIL, API_KEY(CRITICAL), SSN}
    P->>INJ: screen(text)
    INJ-->>P: InjectionFinding {QUARANTINED, 3 categories}
    P->>P: decision = quarantine (injection + critical PII)
    P->>Q: quarantine(provenance, reason, raw, findings, doc)
    Q->>S: save(QuarantineRecord)
    S-->>Q: ok
    Q-->>P: QuarantineRecord
    P-->>C: PipelineResult {QUARANTINED, quarantine_id}
```
