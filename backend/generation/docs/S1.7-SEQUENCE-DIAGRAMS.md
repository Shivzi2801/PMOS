# S1.7 — Context Assembly — Sequence Diagrams

All diagrams use Mermaid. They describe runtime interactions only; no component
performs network or LLM I/O.

## 1. Happy path — `build_prompt` (end to end)

```mermaid
sequenceDiagram
    autonumber
    participant Caller as Caller (Wave 2 boundary)
    participant ASM as ContextAssembler
    participant DUP as DeduplicationFilter
    participant ACL as ACLFilter
    participant RANK as ContextRanker
    participant BUD as TokenBudget
    participant PB as PromptBuilder
    participant MET as AssemblyMetrics

    Caller->>ASM: build_prompt(result, acl, template_name, metrics)
    ASM->>ASM: _validate(result)
    ASM->>DUP: apply(chunks)
    DUP-->>ASM: kept, dropped(dups)
    ASM->>MET: duplicate_chunks_removed
    ASM->>ACL: apply(kept)
    ACL-->>ASM: kept, dropped(acl)
    ASM->>MET: acl_filtered_chunks
    ASM->>RANK: rank(kept)
    RANK-->>ASM: ranked (score desc, id tie-break)
    loop each ranked chunk
        ASM->>BUD: count(chunk.content)
        BUD-->>ASM: token_cost
        alt fits remaining budget
            ASM->>ASM: assign marker [n], record ContextChunk + CitationRecord
        else does not fit
            ASM->>ASM: drop chunk id (continue)
        end
    end
    ASM->>MET: selected_chunks, used_context_tokens
    ASM-->>ASM: ContextPackage
    ASM->>PB: build(context, template_name)
    PB->>BUD: count(message) per message
    BUD-->>PB: token estimates
    PB-->>ASM: PromptPackage
    ASM->>MET: estimated_prompt_tokens
    ASM-->>Caller: PromptPackage
```

## 2. Token budget enforcement (greedy fill detail)

```mermaid
sequenceDiagram
    autonumber
    participant ASM as ContextAssembler._enforce_budget
    participant BUD as TokenBudget

    ASM->>BUD: available_for_context
    BUD-->>ASM: A
    Note over ASM: used = 0, marker_index = 1
    loop for chunk in ranked
        ASM->>BUD: count(chunk.content)
        BUD-->>ASM: cost
        alt used + cost > A
            Note over ASM: drop chunk id, keep scanning<br/>(smaller later chunks may fit)
        else
            Note over ASM: select; marker = [marker_index]<br/>used += cost; marker_index += 1
        end
    end
    ASM-->>ASM: selected, citations, dropped, used
```

## 3. Empty context decision

```mermaid
sequenceDiagram
    autonumber
    participant ASM as ContextAssembler
    participant Caller

    ASM->>ASM: selected = [] after dedup/ACL/budget
    alt allow_empty_context == False
        ASM-->>Caller: raise EmptyContextError
    else allow_empty_context == True
        ASM-->>Caller: ContextPackage(is_empty=True)
    end
```

## 4. Custom prompt template registration

```mermaid
sequenceDiagram
    autonumber
    participant Caller
    participant PB as PromptBuilder
    participant ASM as ContextAssembler

    Caller->>PB: register_template("terse", factory)
    Caller->>ASM: ContextAssembler(budget, prompt_builder=PB)
    Caller->>ASM: build_prompt(result, template_name="terse")
    ASM->>PB: build(context, "terse")
    alt template known
        PB-->>ASM: PromptPackage(template_name="terse")
    else unknown
        PB-->>ASM: raise PromptTemplateError
    end
    ASM-->>Caller: PromptPackage / error
```
